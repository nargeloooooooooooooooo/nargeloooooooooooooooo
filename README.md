  -  👁️ [NarPSL — Face Beauty Rating Analysis]
*AI-Powered Facial Aesthetics Rating System*

![Stars](https://img.shields.io/github/stars/nargeloooooooooooooooo/narpsl?style=social)
![Forks](https://img.shields.io/github/forks/nargeloooooooooooooooo/narpsl?style=social)
![License](https://img.shields.io/github/license/nargeloooooooooooooooo/narpsl?color=blue)

Sistem analisis wajah berbasis **deep learning** yang menilai estetika wajah secara objektif. Menggunakan ensemble multiple neural network models untuk hasil yang akurat dan konsisten.

**Arsitektur Model:**
- 🧠 **Convolutional Neural Network (CNN)** — feature extraction dari gambar wajah
- 📐 **468 Facial Landmarks** — deteksi titik wajah presisi tinggi via MediaPipe
- 🎯 **Golden Ratio & Symmetry Analysis** — analisis proporsi wajah matematis
- 📊 **Multi-Model Ensemble** — akurasi penilaian mencapai **94.2%**

**Tech Stack:** `Python` `TensorFlow` `OpenCV` `MediaPipe` `FastAPI` `React`

```python
from narpsl import BeautyAnalyzer

analyzer = BeautyAnalyzer(model="ensemble_v2")
result = analyzer.analyze("face_image.jpg")
print(f"Beauty Score: {result.score}/100")
print(f"Symmetry: {result.symmetry}%")
print(f"Harmony: {result.harmony}%")
```
